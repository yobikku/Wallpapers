# Wallpapers
Personal Wallpapers Dump that I like, from Distros, Unixporn, or from [/wg/](https://boards.4chan.org/wg/)

Install
```
cd $HOME/Pictures #cd to your pictures directory 
git clone https://github.com/yobikku/Wallpapers
```
